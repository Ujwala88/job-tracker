# 🚀 Job Application Tracker

A personal job tracking system designed to help manage applications, prepare for interviews, and extract meaningful insights from job descriptions.

---

## 🧠 Why I built this

As a product designer actively applying for roles, I needed more than a spreadsheet.

I wanted a system that:

* Tracks applications and their status
* Stores job descriptions and tailored notes
* Helps prepare for interviews
* Surfaces insights from job requirements

This project is my attempt to build a **data-informed job search companion**.

---

## ✨ Features

### 📊 Application Tracking

* Add and manage job applications
* Track statuses (Applied, Interview, Offer, Rejected)
* View summary dashboard

### 🧾 Rich Job Details

* Store full job descriptions
* Save cover letters and notes
* View everything per role in one place

### 🧠 Interview Preparation (in progress)

* Extract key themes from job descriptions
* Align with personal experience
* Highlight areas to focus on before interviews

### 📈 Dashboard Insights

* Total applications
* Active applications
* Interview pipeline
* Offers received

---

## 🛠️ Tech Stack

* **Frontend:** Next.js (App Router)
* **Styling:** Tailwind CSS
* **Backend:** Prisma ORM
* **Database:** SQLite (local)
* **Language:** TypeScript

---

## ⚙️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/Ujwala88/job-tracker.git
cd job-tracker
```

### 2. Install dependencies

```bash
npm install
```

### 3. Set up the database

```bash
npx prisma db push
```

### 4. Run the app

```bash
npm run dev
```
